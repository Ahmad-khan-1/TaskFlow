# TaskFlow

A full-stack todo application that helps users organize their tasks with priority 
levels, categories, and due dates. Built as a learning project to understand 
production-level Next.js architecture — from database design to authentication 
to deployment.

## Features
- User authentication (signup/login) with NextAuth.js
- Create, edit, delete, and complete tasks
- Priority levels (low, medium, high)
- Categories and due date tracking
- Filter and sort functionality
- Protected routes — each user only sees their own tasks

## Tech Stack
- **Frontend:** Next.js 14 (App Router), Tailwind CSS
- **Backend:** Next.js API Routes, Server Components
- **Database:** MongoDB with Mongoose
- **Authentication:** NextAuth.js (Credentials Provider)
- **Deployment:** Vercel
